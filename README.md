# Shufl

Shufl is a card shuffling application.

## Installation

#### 1. Clone the repo

`
$git clone https://github.com/josharagon/shufl.git
`
#### 2. CD into the new directory

`
$cd shufl
`
#### 3. Install Dependencies

`
npm i 
`
#### 4. Start application

`
npm start
`
